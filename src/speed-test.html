Як працює тест швидкості?

Сервіс Speedtest автоматично підбирає найближчий сервер L4 для максимально точних результатів.

Ми вимірюємо три ключові показники вашого інтернет-з'єднання:

  • Швидкість завантаження

    Швидкість отримання даних з інтернету на ваш пристрій. Важлива для перегляду відео, завантаження файлів та веб-серфінгу.

  • Швидкість віддачі

    Швидкість передачі даних з вашого пристрою в інтернет. Критична для відеодзвінків, стрімінгу та завантаження файлів в хмару.

  • Затримка (Ping)

    Час, який потрібен для зв'язку між вашим пристроєм і сервером. Особливо важливо для онлайн-ігор та відеоконференцій.
